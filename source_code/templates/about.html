About the Project

With the increasing connectivity of modern vehicles, cybersecurity has become a critical concern. The Controller Area Network (CAN), which enables communication between vehicle components, is vulnerable to cyberattacks that can manipulate vehicle behavior, leading to safety risks. Our Intrusion Detection System (IDS) is developed to detect, prevent, and mitigate these attacks in real time. By leveraging deep learning, transfer learning, and traditional machine learning techniques, our system enhances security, efficiency, and adaptability in automotive networks.

Project Methodology

Our approach involves a hybrid AI model that combines Convolutional Neural Networks (CNN) for spatial feature extraction and Long Short-Term Memory (LSTM) networks for temporal pattern analysis. Additionally, we integrate Transfer Learning, which allows the system to adapt to new attack patterns with minimal retraining. The data preprocessing phase includes data cleaning, normalization, feature selection, and augmentation to improve model accuracy. The system is evaluated using real-world CAN datasets to ensure high detection accuracy and low false alarm rates.

Advantages

  • High Detection Accuracy (99%+) – Effectively detects multiple attack types.
  • Real-Time Monitoring – Identifies intrusions instantly for proactive security.
  • Adaptive Learning – Uses Transfer Learning to recognize new cyber threats.
  • Lightweight & Efficient – Reduces computational overhead by 65%.
  • Seamless Integration – Can be deployed in modern and next-gen vehicles.

System Requirements

  • Hardware: Minimum 8GB RAM, GPU (for training), and a CAN data interface.
  • Software: Python, TensorFlow/Keras, Scikit-learn, and OpenCV (for data preprocessing).
  • Dataset: Car-Hacking Dataset with attack logs for model training and evaluation.
  • Deployment: Supports integration with ECUs (Electronic Control Units) and automotive security frameworks.
